# Работа со структурами в C++ 13 вариант

## Описание

В работе реализована программа на языке C++, выполняющая операции над массивом структур, описывающих музыкальные альбомы.

Каждый альбом содержит:

* название;
* исполнителя;
* стиль;
* год выпуска;
* общую длительность;
* список композиций (название и длительность).

---

## Используемые типы данных

### Перечисляемый тип

```cpp
enum styles {
    Funk,
    Soul,
    Jazz,
    Rock
};
```

Используется для хранения стиля альбома.

### Структуры

**song** — описание композиции:

* `name` — название;
* `duration` — длительность.

**album** — описание альбома:

* `name` — название;
* `artist` — исполнитель;
* `style` — стиль (enum);
* `year` — год выпуска;
* `total_duration` — суммарная длительность;
* `songs_number` — количество композиций;
* `songs` — указатель на массив структур `song`.

---

## Реализованные функции

### Инициализация массива

```cpp
album* init();
```

Создаёт массив из 20 альбомов и заполняет его данными внутри программы.

---

### Фильтрация по стилю

```cpp
album* filter_funk_soul(album* arr, int& size_filtered);
```

Формирует новый массив, содержащий только альбомы со стилями Funk и Soul.

---

### Сортировка

```cpp
void bubble_sort_by(album* arr, int size, bool (*cmp)(const album&, const album&));
```

Реализована сортировка методом пузырька с использованием функции-компаратора.

Пример компаратора:

```cpp
bool cmp_years(const album& lhs, const album& rhs);
```

---

### Вывод данных

```cpp
void print_album(const album& a);
```

Выводит информацию об одном альбоме.

```cpp
void print_array(album* arr, int size);
```

Выводит массив альбомов.

---

### Поиск трёх самых длинных альбомов

```cpp
void print_top3_longest(album* arr, int size);
```

Сортирует массив по длительности и выводит три альбома с максимальной длительностью.

---

### Изменение данных

```cpp
void edit_album(album& a);
```

Позволяет изменить название альбома и год выпуска.

---

### Фильтрация по исполнителю

```cpp
album* filter_linkin_park(album* arr, int& size_filtered);
```

Формирует массив альбомов исполнителя Linkin Park.

---

## Использование в программе

В функции `main`:

* создаётся исходный массив;
* выполняется фильтрация по стилю;
* выполняется сортировка по году;
* выводятся отфильтрованные данные;
* выводятся три самых длинных альбома;
* выполняется изменение одного альбома;
* формируется массив альбомов Linkin Park и выводится;
* освобождается динамическая память.
